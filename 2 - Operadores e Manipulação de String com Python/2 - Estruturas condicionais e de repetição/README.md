## Identa√ß√£o e blocos

Identar c√≥digo √© uma forma de manter o c√≥digo fonte mais leg√≠vel e manuten√≠vel. Mas em Python ela exerce um segundo papel, atrav√©s da indenta√ß√£o o interpretador consegue determinar onde um bloco de comando inicia e onde ele termina.

>üí°Existe uma conven√ß√£o em Python, que define as boas pr√°ticas para escrita de c√≥digo na linguagem. Nesse documento √© indicado utilizar 4 espa√ßos em branco por n√≠vel de indenta√ß√£o, ou seja, a cada novo bloco adicionamos 4 novos espa√ßos em branco.

    # Fim do bloco do if

**Isso m√£o funciona em Python ‚ùå**

### if, else e elif

S√£o estruturas utilizadas para repetir um trecho de c√≥digo um determinado n√∫mero de vezes. Esse n√∫mero pode ser conhecido previamente ou determinado atrav√©s de uma express√£o l√≥gica.

No Python, existem 3 estruturas de repeti√ß√£o:
- for-in, for-else, for-range

### for

O comando for √© usado para percorrer um objeto iter√°vel. Faz sentido usar for quando sabemos o n√∫mero exato de vezes que nosso bloco de c√≥digo deve ser executado, ou quando queremos percorrer um objeto iter√°vel. O for segue uma estrutura com 2 partes:

   - Vale ressaltar que essa vari√°vel pode ser nomeada de qualquer maneira. Mas √© recomendado utilizar um nome descritivo, algo que represente uma unidade da sequ√™ncia que ser√° percorrida.

No exemplo abaixo, a estrutura vai se repetir na string `texto` (em Python uma string √© um objeto iter√°vel). Dessa forma, cada elemento da posi√ß√£o ser√° uma letra da frase. 
- Em seguida, vamos verificar se cada letra da `texto` est√° presente (utilizando o operador `in`) na constante `VOGAIS`. Se estiver, essa letra ser√° exibida.
  - O comando `letra.upper()` ir√° deixar cada letra em maisculo para realizar a compara√ß√£o.

#### range
Range √© uma fun√ß√£o built-in do Python, ela √© usada para produzir uma sequ√™ncia de n√∫meros inteiros a partir de um √≠nicio (inclusivo) para um fim (exclusivo). Se usarmos range(0, 1)
